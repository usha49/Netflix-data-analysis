# Netflix Data Analysis

## Project Overview
This project analyzes Netflix's content catalog to uncover insights about their content strategy, including content types, genres, geographical distribution, and growth patterns. The analysis is based on a dataset of approximately 9,000 Netflix titles.

## Analysis Questions Explored

1. **Content Distribution** - How many movies vs TV shows are there?
2. **Genre Popularity** - What is the most common genre?
3. **Geographical Analysis** - Which country produces the most Netflix content?
4. **Growth Trends** - How has Netflix grown over the years?
5. **Directors** - Who are the most frequent directors?

The analysis utilizes Netflix's content catalog data containing information about:
- Title information
- Content type (Movie/TV Show)
- Genres and categories
- Country of origin
